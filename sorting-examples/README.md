## 📊 算法详解

### 冒泡排序 (Bubble Sort)
- **原理**：重复遍历数组，比较相邻元素并交换
- **优化**：如果一轮遍历没有交换，说明已有序
- **适用场景**：教学演示，小规模数据

### 快速排序 (Quick Sort)
- **原理**：分治算法，选择基准元素进行分区
- **优势**：实际应用中性能优异
- **注意**：最坏情况为O(n²)，但可通过随机化避免

### 归并排序 (Merge Sort)
- **原理**：分治算法，递归分解后合并有序子数组
- **优势**：稳定排序，时间复杂度稳定
- **适用场景**：链表排序、外部排序

### 堆排序 (Heap Sort)
- **原理**：利用堆数据结构进行排序
- **优势**：原地排序，空间复杂度O(1)
- **特点**：不稳定排序，但性能稳定
